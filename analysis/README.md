# Analysis

This folder contains scripts and datasets for analyzing translation quality.

## Scripts

| Script | Purpose |
|--------|---------|
| `collect_app_data.py` | Collect translation statistics |
| `run_analysis.py` | Run quality analysis |
| `test_translations.py` | Test translation accuracy |

## Datasets

- `test_dataset.json` - Test cases for translation validation

## Results

- `template.json` - Output template for analysis results

## Usage

```bash
cd analysis
pip install -r requirements.txt
python scripts/run_analysis.py
```

> **Note**: These scripts are for development and quality assurance, not required for the app to function.
